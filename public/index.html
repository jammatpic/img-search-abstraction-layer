Image Search Abstraction Layer

Description:

Pass a search query as a parameter, and this service will return search results. You can optionally specify an offset in order to paginate through the results. The "JSONView" extention for Chrome/Firefox makes the results much more readable. Visit https://img-search-abs-layer.herokuapp.com/latest to see the latest searches.

Example:

https://img-search-abs-layer.herokuapp.com/imgsearch/cats?offset=0
